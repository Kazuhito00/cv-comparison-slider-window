# cv-stacksliderwindow
2枚の画像を重ね合わせて、マウススライドで比較するウィンドウのサンプルです。<br>
![bu19j-mplzq](https://user-images.githubusercontent.com/37477845/99143014-9cf5af80-269d-11eb-9eb0-c872d5a5f74c.gif)

* OpenCV 3.4.2 or later

# Demo
デモの実行方法は以下です。
```bash
python sample.py
```
